Python Console Calculator

Description

A simple command-line calculator built with Python. It supports basic arithmetic operations and handles common errors such as division by zero and invalid operators.

Features

- Addition ("+")
- Subtraction ("-")
- Multiplication ("*")
- Division ("/")
- Modulus ("%")
- Power ("**")
- Error handling for invalid input and division by zero
- Option to quit anytime by entering "q"

Requirements

- Python 3.x

How to Run

1. Save the file as "calculator.py".
2. Open a terminal or command prompt.
3. Run the following command:

python calculator.py

Example

=== Python Calculator ===
Operators: + - * / % ** (power)
Type 'q' at any time to quit.

Enter first number: 10
Enter operator (+ - * / % **): *
Enter second number: 5

Result: 10.0 * 5.0 = 50

Error Handling

- Prevents division by zero.
- Displays an error for unsupported operators.
- Allows the user to exit the program by entering "q".
